# C. shortcut

```
1. spacebar: 타이머 시작/중지
2. command + 1: 5m
3. command + 2: 10m
4. command + 3: 25m
5. esc: switch to transparent mode
6. 숫자키: 시간 설정(in minute, transparent mode only)
```
